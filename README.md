# Ruby NACSIS CAT

Ruby用 NACSIS-CAT 接続ライブラリ です。

SEARCH,RETRIEV,INSERT,UPDATE,DELETE を実装予定です。 

## Installation

Add this line to your application's Gemfile:

    gem 'ruby_nacsiscat'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby_nacsiscat
